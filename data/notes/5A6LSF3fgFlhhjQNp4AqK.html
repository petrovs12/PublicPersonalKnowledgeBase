  1. Find current function Q_{\theta^{(t)}}=Q(\theta,\theta^{(t)}) .

Motivation:

If the value of the parameters θ\thetaθ is known, usually the value of the latent variables ZZZ can be found by maximizing the log-likelihood over all possible values of Z\ZZ , either simply by iterating over ZZZ or through an specialized algorithm for the specific generative structure. For example, if we have an HMM, the MLE estimates for it's latent variables can be estimated by the science.CS.algos.dynamicProgramming.Viterby Algorithm (Private). Conversely, if we know the value of the latent variables ZZZ , we can find an MAP estimate of θ\thetaθ by maximizing the log likelyehood. This suggests an iterative algorithm, in the case where both θ\thetaθ and ZZZ are unknown .

  1. First, initialize the parameters θ\thetaθ to some random values.
  2. Compute the probability of each possible value of ZZZ, given θ\thetaθ.
  3. Then, use the just-computed values of ZZZ to compute a better estimate for the parameters θ\thetaθ.
  4. Iterate steps 2 and 3 until convergence. The algorithm as just described monotonically approaches a local minimum of the cost function.
