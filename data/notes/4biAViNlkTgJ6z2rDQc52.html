Consistent Hashing

Motivation:

We have a stream of BLOB objects, and want to assign them to nnn servers in a 'uniform' way. We also want to ensure that if a server is added or removed, the 'movement' of objects is not too much. In particular, if we add or remove a server we want only about 1/n1/n1/n) of the objects to move around.

Idea: Use a hash function that assigns both objects and servers to points in the unit circle. So f:A−>[0,2π]f:A->[0,2\pi]f:A−>[0,2π], for both servers and objects.
