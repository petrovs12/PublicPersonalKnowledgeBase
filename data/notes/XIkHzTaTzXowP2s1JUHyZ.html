Performance monitoring and tracking

As part of the model serving infrastructure in a given system, we could have a Load Balancer, or more generally, a dispatcher, which, for each incoming request, determines which model to serve.

The disparcher then returns the modelid, and we keep track which model id/configuration is served for each request.

Then after the data is in DWH, we can use Tests, AB Testing, Tests

We can also use Reinforcement Learning in the dispatcher in order figure out what to send.
