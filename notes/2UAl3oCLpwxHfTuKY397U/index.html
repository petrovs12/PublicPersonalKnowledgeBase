Baby Shopping List

Watch some baby first aid videos/take a course. Get first aid and grooming kit for newborn. Also get some special baby laundry detergent and wash new everything new before wearing. Idk if needed, but my partner did it.

  1. Baby stroller- not much opinion, you can buy used, just test it, look so it has big wheels as it's easier to handle. Also if you're driving, be careful if buying so it's easy to put a base on the car. I think there are different standards and so on. For us we put it without the base.

  1. Bed - We used Snoo Smart Sleeper . By far the best item on the list if you can get it, highly recommended. Will send some sleeping data separately. You can also look at this article. Also consider buying some baby swaddles. We bought some, but practically used only the ones w/ the snoo.

  2. Books - we read very little, but I do recommend Emily Oster's books. We are using a lot of the advice there. It appeals to me cause it's short and it has a lot of meta-analyses of papers and help you ease your anxiety specifically about breastfeeding and so. She now has a blog, which is pretty nice too. I went trough first 5 chapters of Baby 411, it's great, then used rest for reference

  1. Dedicated white noise machine is a great one.0ww

  1. In-ear or contactless thermomether. Bonus: for boys for me it was super helpful to use the in-ear thermometer cap as a proxy for the bellow item the penis during diaper change.

  2. For boys only. Helps with diaper change stress (you don't ha).

  3. Anti-smell diaper bin. Helps a lot with the smell and have to change the inside bag only 1-2 times a week.

  4. Diaper changing mat. Diapers, 99% water baby wipes, diaper changing non-reusable pads. Put those to subscription.

  5. Pacifiers- buys several types to try

  6. Breast pump- not sure, we used mostly a manual one, as it's easier to control power, but had a electrical one as well.

  7. Baby bottle sanitizer (we had Phillips).

  8. Baby bottle warmer, just picked a random one.

  9. Nasal Aspirator- I think this kind is quite good. You have some in the grooming kit as well, but for us we needed another kind...

  10. Baby carrier and maybe winter/rain protector for it. We're not using the carrier too much, but that's cause I'm lazy and it was super cold. Probably go for ergobaby or so.

  11. Maybe a baby bathtub. We used it, but maybe you can just do it in the sink

  12. If supplementing, consider looking for low flow bottles so it's harder to reject breast.

  13. Consider some contraption like this in case baby is fussy on breast. We tried it but was a bit annoying to use. But if you're commited it probably helps...

  14. Baby doses of saline solution. Honestly haven't used, but meh.

  15. Liquid Vitamin D and some topical baby oil.

  16. Consider putting some matress topper on your bed, cause you know.

  17. Buy some clothes as needed.
