Exploring w/ dfs


seen = set()
def explore(V:List[Int],E:Dict[Int,Set[int]],v):
    if v in seen:  #optionally!!!
        return
    seen.add(v)
    previsit(v) # optionally! use if need to detect cycles, other stuff 
    for w in E[v]:
        explore(V,E,w)
    postvisit(v) # optionally! useful to keep track of cycles and other stuff

Intuition: Explore labyrinth: need some rope (stack) and piece of chalk(visited set). Go along the rightmost path without chalk marks (dfs ordering), and return when you reach a dead end.
